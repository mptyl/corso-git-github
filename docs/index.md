# Corso Git & GitHub

Benvenuto al corso pratico di **Git e GitHub**!

## Obiettivi del corso

- Imparare a usare **Git** per il versionamento del codice
- Collaborare su **GitHub** con pull request, code review e branch strategy
- Gestire conflitti, rollback e hotfix
- Pubblicare documentazione con **GitHub Pages**

## Lezioni

| # | Argomento | Tempo |
|---|-----------|-------|
| 0 | Setup: Installazione e configurazione | 15 min |
| 1 | Creare il progetto | 15 min |
| 2 | Unirsi al progetto | 10 min |
| 3 | Worktree e prima feature | 20 min |
| 4 | Worktree parallelo | 20 min |
| 5 | La prima Pull Request | 15 min |
| 6 | Conflitto di merge | 25 min |
| 7 | Rollback e correzioni | 20 min |
| 8 | Hotfix di produzione | 15 min |
| 9 | Documentazione e GitHub Pages | 25 min |

---

> **Nota:** Segui le lezioni in ordine — ogni lezione costruisce su quella precedente.
